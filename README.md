# react-seamless-scroll

>一个基于 `requestAnimationFrame` 的`react`自动无缝滚动组件！

![示例](./egg.gif)

## 快速开始

---

- `npm install react-seamless-scroll -S`
- `import ReactSeamlessScroll from 'react-seamless-scroll'`;

## 使用方法

---

如下props:

Option               | default       | Description
---------------------|---------------|-----------------------------------------------
`speed`            |  60        | 每秒/px
`reverse`   |  false         | 是否逆向滚动
`mode`            |  vertical         | 滚动方向
